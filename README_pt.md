Aprenda sobre Alta Disponibilidade com Keepalived. Este módulo abrange Conceitos e Instalação do VRRP, Configuração do Nó Mestre, Configuração do Nó de Backup, Teste de Failover, Failover com Consciência de Serviço. Você dominará estas habilidades essenciais do Linux através de laboratórios práticos e desafios do mundo real.
